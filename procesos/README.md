# Mapas de Proceso y Flujos Lean

Este directorio contiene toda la arquitectura operativa del sistema de auditoría inteligente.

## Documentos incluidos
- Mapa de flujo de valor (VSM) del proceso manual original
- VSM del proceso Lean optimizado
- Flujo de inspección automática
- Flujo de gestión de excepciones
- Proceso de escalamiento a auditoría
- Flujo estándar para corrección y retroalimentación

El propósito es eliminar cuellos de botella, reducir tiempos y garantizar que cada documento fluya con calidad desde su entrada hasta su archivado final.
