# Scripts y Automatizaciones

En este directorio se encuentran los programas responsables de inspeccionar, validar y clasificar documentos.

## Ejemplos de scripts
- Validación de ID en nombre vs. ID en contenido
- Detección de PDFs vacíos o sin texto
- Clasificación automática por tipo de documento
- Generación de reportes en CSV o Excel
- Simuladores de fallos para entrenamiento

Estos scripts son el núcleo del Poka-Yoke digital del sistema.
