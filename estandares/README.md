# Estándares del Sistema Lean de Auditoría Inteligente

Este directorio contiene los estándares operativos que gobiernan el sistema. Su objetivo es eliminar variabilidad, reducir errores y asegurar que cada documento cumpla con los criterios de calidad definidos.

## Contenido
- Convenciones de nombres de archivo
- Esquema de metadatos
- Formato obligatorio para contratos y documentos críticos
- Requerimientos técnicos mínimos
- Criterios de validación automatizada (Poka-Yoke)
- Reglas de control de versiones y trazabilidad

Los estándares son de cumplimiento obligatorio para garantizar integridad, consistencia y auditoría continua.
