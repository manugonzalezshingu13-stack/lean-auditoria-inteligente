# Auditorías y Registros de Control

Este directorio almacena evidencia histórica, análisis de errores y reportes estructurados que permiten seguir la evolución del sistema Lean.

## Contenido
- Bitácoras de auditoría
- Errores detectados por el sistema automatizado
- Auditorías trimestrales
- Reportes de causas raíz (RCA)
- Pruebas de mejora continua (Kaizen)
- Casos especiales y documentación de fallos críticos

Este espacio permite rastrear decisiones, medir impacto y sostener la mejora continua mediante evidencia real.
