# Dashboard de Indicadores Clave del Sistema

Este dashboard resume el desempeño global del sistema de auditoría.

## Indicadores principales
- Tiempo promedio por documento (min)  
- Errores críticos detectados  
- Porcentaje de aprobaciones automáticas  
- Documentos enviados a OCR  
- Documentos rechazados  
- Tendencia semanal de errores  
- Tiempo total ahorrado vs proceso manual  

## Visualizaciones recomendadas
- Histograma de tiempos
- Gráfica de barras por categoría de error
- Tendencia semanal express (línea)

Este dashboard es suficiente para medir impacto y tomar decisiones operativas.
