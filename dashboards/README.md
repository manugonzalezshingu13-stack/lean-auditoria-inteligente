# Dashboards y Visualizaciones

Este directorio contiene visualizaciones que permiten entender el comportamiento real del sistema.

## Indicadores incluidos
- Tiempos promedio de procesamiento
- Cantidad y tipo de errores detectados
- Comparativa antes/después del sistema Lean
- Porcentaje de validaciones automáticas vs. manuales
- Tendencias de calidad por semana o mes

Los dashboards sostienen la toma de decisiones y muestran el impacto real de la automatización.
