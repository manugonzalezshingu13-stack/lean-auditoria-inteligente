# Documentación General del Proyecto

Este directorio es la columna vertebral conceptual del sistema Lean aplicado a auditoría documental.

## Contenido
- Introducción a Lean para procesos digitales
- Jidoka y su aplicación en la revisión automática de archivos
- Flujo continuo sin lotes
- Estandarización operativa
- Poka-Yoke digital para prevenir errores
- Ciclo Kaizen para mejorar el sistema
- Manual de entrenamiento para nuevos operadores

Aquí se explica el “por qué” y el “cómo” del sistema, permitiendo que otros lo adopten, lo escalen o lo mejoren.
