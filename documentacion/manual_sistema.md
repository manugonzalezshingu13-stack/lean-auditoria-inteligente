# Manual del Sistema Lean de Auditoría Inteligente

## 1. Qué es este sistema
Un marco modular para auditar procesos sin desperdicio y con evidencia verificable.

## 2. Cómo funciona
- Checklists estandarizadas  
- Plantillas maestras  
- Metodología unificada  
- Automatización ligera opcional

## 3. Roles
- Auditor  
- Responsable del proceso  
- Revisor  
- Dueño del repositorio  

## 4. Flujo
1. Preparación  
2. Auditoría  
3. Documentación  
4. Acción  
5. Mejora continua
