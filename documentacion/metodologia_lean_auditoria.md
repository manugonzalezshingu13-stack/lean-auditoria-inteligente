# Metodología Lean para Auditorías Inteligentes
Versión: 1.0

Esta metodología fusiona:
- Jidoka (detener, detectar, corregir)
- Genchi Genbutsu (ir a la fuente)
- Estandarización
- Ciclo PHVA
- Automatización ligera

## 1. Preparación
1. Ver el proceso en vivo  
2. Identificar flujo real  
3. Revisar documentación previa  
4. Elegir la checklist adecuada  

## 2. Ejecución
1. Realizar recorrido  
2. Identificar desviaciones  
3. Registrar evidencia  
4. Entrevistar operadores  

## 3. Análisis
1. Clasificar hallazgos  
2. Buscar causas raíz  
3. Estimar impacto  

## 4. Acciones Lean
- Simplificar  
- Estandarizar  
- Automatizar tareas repetitivas  
- Documentar nueva condición
