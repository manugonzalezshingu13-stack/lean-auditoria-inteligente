# Checklist de Validación Básica del Documento

Este checklist se aplica a cada archivo que ingresa al sistema Lean de Auditoría Inteligente.

## 1. Validación del nombre del archivo
[ ] El nombre sigue la convención: TIPO_ID.pdf  
[ ] El ID contiene solo números  
[ ] El tipo de documento es válido (CONTRATO, ACTA, SOLICITUD, etc.)  
[ ] No hay espacios, caracteres especiales ni acentos

## 2. Validación del contenido interno
[ ] El archivo contiene texto legible  
[ ] El ID del nombre aparece dentro del contenido  
[ ] El contenido no está vacío  
[ ] El formato legal es correcto (si aplica)  

## 3. Validación técnica
[ ] El archivo no está corrupto  
[ ] El tamaño del archivo es mayor a 0 KB  
[ ] La conversión OCR fue exitosa (si aplica)

## 4. Clasificación y resultado final
[ ] Documento aprobado  
[ ] Documento rechazado por inconsistencia ID  
[ ] Documento enviado a revisión manual  
[ ] Documento requiere OCR  
[ ] Documento enviado a auditoría crítica

Cada marca incorrecta debe documentarse en la bitácora de auditoría correspondiente.
