# Checklists de Auditoría

Este directorio reúne las listas de verificación utilizadas para revisar documentos antes de aprobarlos, rechazarlos o enviarlos a revisión manual.

## Tipos de checklists
- Verificación de nombre y metadatos
- Coherencia entre texto interno y nombre del archivo
- Validación de formato obligatorio
- Detección de PDF vacíos, corruptos o ilegibles
- Confirmación de requisitos legales
- Revisiones incrementales para auditoría Lean

Cada checklist es parte esencial del sistema Jidoka: detener el proceso cuando aparece un defecto, corregir la causa y documentar la mejora.
