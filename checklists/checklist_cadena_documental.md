# Checklist de Auditoría de Cadena Documental
Versión: 1.0

## 1. Identificación del documento
[ ] Nombre del archivo correcto  
[ ] Estructura de nombre según estándar  
[ ] Código único presente  

## 2. Integridad del contenido
[ ] El código coincide dentro del texto  
[ ] No hay páginas faltantes  
[ ] No hay contenido ilegible u OCR fallido  

## 3. Riesgos detectados
[ ] Manipulación sospechosa  
[ ] Fechas inconsistentes  
[ ] Falta de firmas
